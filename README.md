# 🖌️ Virtual Painter App with Hand Gestures

A fun and interactive **Virtual Painter** built using **OpenCV** and **MediaPipe**, controlled entirely by **hand gestures**! No mouse or keyboard required — just your webcam and fingers. 🎨✋

---

## 📸 Features

- ✍️ **Draw** with your index finger
- 🎨 **Change color** by hovering over color buttons
- 📏 **Adjust brush thickness** dynamically
- 🗑️ **Clear canvas** with a fist gesture
- ⚡ Real-time gesture detection using **MediaPipe**

---

## 🛠️ Tech Stack

- **Python**
- **OpenCV**
- **MediaPipe**
- **NumPy**
